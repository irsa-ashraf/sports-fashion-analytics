## Project Overview

This project analyzes the data-driven intersection of sports and fashion, focusing on three major trends:

1. **The "Challengers" Effect** - How Zendaya's 2024 film sparked the tenniscore fashion explosion
2. **Women's Sportswear Growth** - $92B â†’ $250B market trajectory (2023-2033)
3. **World Cup 2026 Predictions** - Forecasting soccer jersey trends in the US

### Key Findings

- **Skort sales +134% YoY** after Challengers release
- **Adult tennis lesson searches +245%** post-film
- **$42.5M media value** generated by Zendaya's press tour
- **+40% jersey sales spike** expected during World Cup 2026

### Prerequisites
- Python 3.11+
- pip

### Installation

1. **Clone the repository**
```bash
git clone https://github.com/YOUR-USERNAME/sports-fashion-analytics.git
cd sports-fashion-analytics
```

2. **Install dependencies**
```bash
pip install -r requirements.txt
```

3. **Run data processing pipeline**
```bash
cd src/data_processing
python process_data.py
```

## Data Sources

### Primary Sources
- **Google Trends**: Search volume data for tenniscore, soccer jerseys
- **Clearpay/Afterpay**: E-commerce sales growth data
- **WeArisma**: Social media analytics and media value
- **Market Research Reports**: Industry forecasts and projections

### Data Coverage
- **Time Range**: January 2023 - February 2026
- **Geography**: Primarily US market with global context
- **Demographics**: Gen Z, Millennials, Gen X, Baby Boomers


### Key Metrics Display
- Real-time KPIs and growth rates
- Demographic segmentation
- Year-over-year comparisons
- Media value tracking

### Data Export
- Download processed datasets as CSV
- Interactive filtering and exploration
- Jupyter notebook integration
